INTRODUCTION
-------------------
THIS MODEL PREDICTS THE AMOUNT OF CO2 EMISSIONS FROM NEW MOTOR VEHICLES TO ENSURE GOVERNMENT COMPLIANCE.

REQUIREMENTS
-------------------
-MICROSOFT WINDOWS 10 x64
-R x64 4.0.2 OR HIGHER
-RSTUDIO
-DATASET OF VEHICLE EMISSIONS (OBTAINED FROM KAGGLE)
-THE FOLLOWING R LIBRARIES:
	-CAR
	-DPLYR
	-GGPLOT2
	-TIDYR
	-TM

INSTALLATION
-------------------
IF ANY OF THE R LIBRARIES ARE NOT PRESENT, USE THE FOLLOWING COMMAND TO INSTALL THEM:
install.packages('NAME OF THE LIBRARY')
