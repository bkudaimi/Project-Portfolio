INTRODUCTION
-------------------
THIS MODEL IDENTIFIES WHICH DELIVERIES FROM AN ONLINE RETAILER WILL BE LATE SO THE RETAILER CAN REMEDY THE PROBLEM ACCORDINGLY.

REQUIREMENTS
-------------------
-MICROSOFT WINDOWS 10 x64
-R x64 4.0.2 OR HIGHER
-RSTUDIO
-THE FOLLOWING R LIBRARIES:
	-CAR
	-DPLYR	
	-GGPLOT2
	-TIDYR
	-TM

INSTALLATION
-------------------
IF ANY OF THE R LIBRARIES ARE NOT PRESENT, USE THE FOLLOWING COMMAND TO INSTALL THEM:
install.packages('NAME OF THE LIBRARY')
