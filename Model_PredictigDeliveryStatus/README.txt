INTRODUCTION
-------------------
THIS MODEL IDENTIFIES WHICH DELIVERIES FROM AN ONLINE RETAILER WILL BE LATE SO THE RETAILER CAN REMEDY THE PROBLEM ACCORDINGLY.

REQUIREMENTS
-------------------
-MICROSOFT WINDOWS 10 x64
-PYTHON 3.9.4 OR HIGHER
-JUPYTER NOTEBOOK
-THE FOLLOWING PYTHON LIBRARIES:
	-COLLECTIONS
	-IMBLEARN
	-MATPLOTLIB
	-NUMPY
	-PANDAS
	-PYLAB
	-SCIPY
	-SEABORN
	-SKLEARN
	-STATISTICS
	-WARNINGS

INSTALLATION
-------------------
IF ANY OF THE PYTHON LIBRARIES ARE NOT PRESENT, USE THE FOLLOWING COMMAND TO INSTALL THEM:
pip install [NAME OF THE LIBRARY]
ENSURE THE LATEST VERSION OF PIP IS INSTALLED.
