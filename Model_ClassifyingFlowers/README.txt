THIS CLASSIFIER ANALYZES A PICTURE CONTAINING A FLOWER AND IDENTIFIES THE SPECIES OF FLOWER.

REQUIREMENTS
-------------------
-MICROSOFT WINDOWS 10 x64
-PYTHON 3.9.4 OR HIGHER
-JUPYTER NOTEBOOK
-THE FOLLOWING PYTHON LIBRARIES:
        -KERAS
        -MATPLOTLIB
        -OS
        -PANDAS
        -SPLITFOLDERS
        -TENSORFLOW
        -WARNINGS

INSTALLATION
-------------------
IF ANY OF THE PYTHON LIBRARIES ARE NOT PRESENT, USE THE FOLLOWING COMMAND TO INSTALL THEM:
pip install [NAME OF THE LIBRARY]
ENSURE THE LATEST VERSION OF PIP IS INSTALLED.
