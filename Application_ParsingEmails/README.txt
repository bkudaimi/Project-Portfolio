INTRODUCTION
-------------------
THIS APPLICATION CONVERTS EMAILS IN HTML FORMAT TO READABLE TEXT THEN SORTS EACH EMAIL BY ITS PART (FROM, TO, TITLE, ETC).

REQUIREMENTS
-------------------
-MICROSOFT WINDOWS 10 x64
-PYTHON 3.9.4 OR HIGHER
-JUPYTER NOTEBOOK
-THE FOLLOWING PYTHON LIBRARIES:
	-BS4
	-CHARDET
	-COLLECTIONS
	-DATETIME
	-DATEUTIL
	-EMAIL
	-JSON
	-OS
	-PANDAS
	-PATHLIB
	-PYSPARK
	-S3FS
	-ZIPFILE
	
INSTALLATION
-------------------
IF ANY OF THE PYTHON LIBRARIES ARE NOT PRESENT, USE THE FOLLOWING COMMAND TO INSTALL THEM:
pip install [NAME OF THE LIBRARY]
ENSURE THE LATEST VERSION OF PIP IS INSTALLED.
