INTRODUCTION
-------------------
THIS MODEL PREDICTS THE PROBABILITY OF AN INTERNATIONAL STUDENT GAINING ADMISSION TO AN AMERICAN GRADUATE SCHOOL (CAN BE ADAPTED FOR DOMESTIC STUDENTS AS WELL).

REQUIREMENTS
-------------------
-MICROSOFT WINDOWS 10 x64
-PYTHON 3.9.4 OR HIGHER
-JUPYTER NOTEBOOK
-THE FOLLOWING PYTHON LIBRARIES:
	-MATPLOTLIB
	-NUMPY
	-KERAS
	-PANDAS
	-SEABORN
	-SKLEARN
	-TENSORFLOW
	-WARNINGS

INSTALLATION
-------------------
IF ANY OF THE PYTHON LIBRARIES ARE NOT PRESENT, USE THE FOLLOWING COMMAND TO INSTALL THEM:
pip install [NAME OF THE LIBRARY]
ENSURE THE LATEST VERSION OF PIP IS INSTALLED.
