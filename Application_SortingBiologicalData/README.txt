INTRODUCTION
-------------------
THIS APPLICATION SORTS HPLC DATA IN EXCEL FILES AND OUTPUTS THE DATA AS GRAPHS OF PROTEIN CONCENTRATION OVER TIME.

INPUT: NAME OF A FOLDER CONTAINING PROPRIETARY HPLC DATA EXPORTED AS EXCEL FILES. 
OUTPUT: FIVE BAR CHARTS OF LIPOPEPTIDE PEAK AREAS FOR EACH SAMPLE AND ONE EXCEL FILE CONTAINING THE PEAK AREA VALUES.

REQUIREMENTS
-------------------
-MICROSOFT WINDOWS 10 x64 (WINDOWS 7 x64 FOR OLDER VERSION)
-PYTHON 3.8.4 OR HIGHER
-PYCHARM IDE
-HPLC DATA AS CSV FILES INSIDE A FOLDER.
-THE FOLLOWING PYTHON LIBRARIES:
	-MATPLOTLIB
	-OS
	-PANDAS
	-PATHLIB
	-WARNINGS
	-XLRD (FOR WINDOWS 7 VERSION)

INSTALLATION
-------------------
IF ANY OF THE PYTHON LIBRARIES ARE NOT PRESENT, USE THE FOLLOWING COMMAND TO INSTALL THEM:
pip install [NAME OF THE LIBRARY]
ENSURE THE LATEST VERSION OF PIP IS INSTALLED.
